# ShatExtension

A personal [Mihon](https://mihon.app) extension repository.

## Add to Mihon

1. Enable **Settings → Browse → Show NSFW sources**.
2. Go to **Settings → Browse → Extension repos → Add** and paste:

   ```
   https://raw.githubusercontent.com/Dat-Ngu/ShatExtension/repo/index.min.json
   ```

3. Install from **Browse → Extensions**.

Built and published automatically via GitHub Actions.
